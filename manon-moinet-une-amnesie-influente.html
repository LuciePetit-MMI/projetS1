Manon Moinet, une amnésie influente

Franck Thilliez est l’auteur du roman à succès, La mémoire fantôme. On y retrouve le lieutenant Lucie Hennebelle, sur la piste du Professeur. Elle sera aidée d’une des victimes de cette histoire, Manon Moinet. Malgré son amnésie, Manon est bien décidé à retrouver Le Professeur. Dans ce roman, on peut mettre avant un fléau encore trop peu connu, la pollution numérique.

Lucie Hennebelle est lieutenant de la criminelle à Lille. Un soir pluvieux elle retrouve une femme dans un piteux état. Elle se prénomme Manon Moinet. Après des examens à l’hôpital, Lucie découvre que cette femme est amnésique. Avec de grosses difficultés, elle mène l’enquête pour découvrir ce qu’il lui est arrivé. Plus l’enquête avance et plus ça se complique. Malgré son amnésie antérograde, Manon insiste pour la seconder dans cette enquête. Plus elles avancent et plus Manon sent une vieille histoire refaire surface. Pr est de retour.

Durant cette enquête les deux femmes apprennent à se connaître. Lucie découvre que Manon est incapable de se créer des souvenirs. Afin de pouvoir mener un semblant de vie, elle doit utiliser un Ntech. C’est une tablette équipée d’une seule application, MemoryNode. Cette application est développée pour aider les amnésiques. Elle permet de stocker toutes les actions qu’ils font dans une journée. Prendre son petit-déjeuner. Nourrir son chien. Aller chez le médecin. Faire une nouvelle rencontre. La moindre action est stockée dans le Ntech pour ne pas être oubliée. Cet outil permet aux amnésiques d’avoir un minimum d’indépendance.

Toutes ces données sont stockées sur un serveur. Cela permet d’avoir une trace de ce qu’elle écrit dans son Ntech. Mais attend, c’est quoi un serveur ? Un serveur est un ordinateur capable de fournir un service. Par service, j’entends ici, page web, application, boite mail ou encore base de données. À chaque fois qu’on envoie un mail ou un sms, il transite par un serveur. Chaque document que l’on stock sur le cloud est en réalité stocké sur un serveur. Dans le cas de MemoryNode, ils permettent de stocker les données qu’on entre dans l’application. Il y a aussi une fonction de sauvegarde pour ne pas perdre les anciennes données. Ils synchronisent aussi les données entre le Ntech et un ordinateur. Ceci permet de consulter ses données n’importe où. Ces serveurs sont une grande source de pollution.
